# ISO 9001 — Quality Management System

## 1. Customer Focus

| Requisito | Implementación |
|-----------|----------------|
| **Needs assessment** | Roles AGRICULTOR/MODERADOR adaptados a perfiles de usuario reales |
| **Feedback loops** | Sistema de reportes con estado COMUNIDAD permite retroalimentación entre usuarios |
| **Satisfaction** | Valoración 5 estrellas en recomendaciones de la comunidad |

## 2. Leadership

| Requisito | Implementación |
|-----------|----------------|
| **Quality policy** | CI/CD con gates de calidad: lint → test → build |
| **Roles & responsibilities** | Guards por rol y permisos en cada endpoint |

## 3. Planning

| Requisito | Implementación |
|-----------|----------------|
| **Risk assessment** | Manejo de errores global con HttpException |
| **Quality objectives** | Performance Monitor con alertas >1000ms |

## 4. Support

| Requisito | Implementación |
|-----------|----------------|
| **Resources** | Infraestructura Docker, pool de conexiones PostgreSQL |
| **Competence** | TypeScript strict, validación DTO, tests unitarios |
| **Documentation** | Swagger/OpenAPI, este documento, README principal |

## 5. Operation

| Requisito | Implementación |
|-----------|----------------|
| **Planning & control** | Migraciones Drizzle versionadas, esquema centralizado |
| **Design & development** | Arquitectura modular NestJS, 12 módulos con responsabilidad única |
| **External providers** | MinIO S3-compatible, PostgreSQL manejado vía pool |

## 6. Performance Evaluation

| Requisito | Implementación |
|-----------|----------------|
| **Monitoring** | PerformanceMiddleware, Health check endpoint |
| **Audit** | Historial de cambios en reporte_historial_estado |
| **Analysis** | Reportes JUnit en CI/CD, cobertura de tests |

## 7. Improvement

| Requisito | Implementación |
|-----------|----------------|
| **Non-conformity** | Filtro global de excepciones, logging estructurado |
| **Corrective actions** | Offline-first con sync diferido, retry en operaciones fallidas |
| **Continuous improvement** | CI/CD automatizado, detección de cambios incremental |
